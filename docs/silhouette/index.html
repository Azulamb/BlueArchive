このページについて

生徒の立ち絵をシルエットにしてプロフィールの身長から大まかに比較できるようにしたページです。作るのが大変なので通常衣装のみです。

追加も不定期です。

身長について

大まかにかかとから頭の先までをプロフィールの身長に合わせて拡大縮小しています。膝や腰を曲げていたり帽子や角が大きい生徒もいますが、かかとと頭頂部を大まかに合わせているので体の大きさ合わせはあまり考えていません。あくまで身長の参考にしてください。特にブルアカはヘイローがあるので純粋な身長が見えにくいですが、その感覚を込みにして感じてもらうためヘイローなどは身長に含んでいません。
